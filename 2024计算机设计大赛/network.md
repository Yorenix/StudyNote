# networkx学习





可视化：

spring_layout：中心放射状
circular_layout：在一个圆环上均匀分布节点
random_layout：随机分布节点
shell_layout：节点都在同心圆上





经典图：

1. `davis_southern_women_graph`：Davis南方女子俱乐部图。这个图描述了一个上个世纪70年代美国南方女子社交圈的关系网络。
2. `florentine_families_graph`：佛罗伦萨家族关系图。这个图展示了15世纪佛罗伦萨城的贵族家族之间的婚姻和亲属关系。
3. `les_miserables_graph`：悲惨世界图。这个图展示了维克多·雨果的小说《悲惨世界》中的角色之间的关系。
4. `barbell_graph`：杠铃图。这个图是一个由两个完全图连接而成的图，类似于杠铃的形状。
5. `ladder_graph`：梯形图。这个图是两行节点按照规则连接而成的图，类似于梯子的形状。
6. `star_graph`：星型图。这个图是一个中心节点连接到其他节点的图，类似于星星的形状。
