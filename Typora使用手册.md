# Typora 软件操作技巧清单

## 一、 极速文本样式 (快捷键)
- 一级至六级标题: `Ctrl+1` ~ `Ctrl+6`
- 提升/降低标题级别: `Ctrl+ =` / `Ctrl+ -`
- 加粗: `Ctrl+B`
- 斜体: `Ctrl+I`
- 下划线: `Ctrl+U`
- 删除线: `Alt+Shift+5`
- 行内代码: `Ctrl+Shift+“` (反引号键)
- 高亮文字: `==文字==` (部分主题需手动加等号，暂无默认快捷键)

## 二、 列表操作技巧 (实心/空心/序号)
- 创建无序列表 (实心圆点): `* 空格` 或 `- 空格` 或 `+ 空格`
- 创建有序列表 (数字序号): `1. 空格` (会自动递增)
- 创建任务列表 (空心方框): `- [ ] 空格`
    - 勾选/取消勾选: 直接鼠标点击方框 (或按 `X` 键在源码模式下)
- 列表缩进 (进入子层级): `Tab` 键
- 列表回退 (返回父层级): `Shift+Tab` 键
- 列表换行不中断序号: `Shift+Enter` (软换行)

## 三、 代码与公式插入技巧
- 插入行内代码块: `Ctrl+Shift+“`
- 插入多行代码块: `Ctrl+Shift+K` (或输入 ` ``` ` 后按回车)
- 快速指定代码语言: 在 ` ``` ` 后直接写 `python` 然后按 `Enter`
- 插入行内数学公式: `$` 输入内容后按 `Esc` 退出编辑
- 插入块级数学公式: `$$` 后按 `Enter`

## 四、 表格与引用
- 快速插入表格: `Ctrl+T` (弹窗选择行列数)
- 表格内增加一行: 光标在行尾按 `Ctrl+Enter`
- 表格内删除一行: `Ctrl+Shift+Backspace`
- 引用文字: `> 空格` (连续 `Enter` 两次退出引用)

## 五、 图片与链接的便捷操作
- 插入链接: `Ctrl+K`
- 快速粘贴图片 (自动复制到本地): 直接 `Ctrl+V` 截图/图片
- 插入本地图片: `Ctrl+Shift+I`
- 快速跳转至链接地址: `Ctrl+鼠标左键点击`

## 六、 视图与专注模式
- 切换源代码模式 (查看纯文本): `Ctrl+/`
- 显示/隐藏侧边栏 (文件列表): `Ctrl+Shift+L`
- 显示/隐藏大纲 (目录导航): `Ctrl+Shift+1`
- 专注模式 (当前行高亮): `F8`
- 打字机模式 (当前行居中): `F9`
- 全屏模式: `F11`
- 暗色/亮色主题切换: 菜单栏 `视图` → `主题`

## 七、 查找与替换效率技巧
- 普通查找: `Ctrl+F`
- 查找并替换: `Ctrl+H`
- 快速定位到指定标题: `Ctrl+P` (搜索框内输入 `@` 会只搜索标题)
- 全局搜索整个文件夹: `Ctrl+Shift+F`

## 八、 自定义与配置技巧
- 修改默认图片保存位置: `文件` → `偏好设置` → `图片` → `复制到指定路径`
- 开启打字机/专注模式自动居中: `视图` → `打字机模式`
- 自定义快捷键: `文件` → `偏好设置` → `通用` → `自定义快捷键`
